# Digital-Literacy-Portfolio
The motive of this project is to help people navigate the digital world — from creating professional online profiles, to communicating safely and responsibly online.

- Name: Shrestha Singh

**Project Overview:**

This project focuses on building essential digital skills required for academic and professional success. As a Student Digital Ambassador, the objective is to create a comprehensive Digital Literacy Portfolio that helps peers understand safe, responsible, and effective use of digital tools.

The project includes five key tasks: designing an awareness infographic, creating a professional digital portfolio, exploring coding and collaboration platforms, practising email etiquette, and developing a cybercrime awareness guide.

Overall, this project enhances digital awareness, communication skills, and online safety practices among students.

**Module Summaries:**

- Module 1:
Concept of digital literacy and its importance in academic and professional life.
- Module 2:
Creating a professional online identity using platforms like GitHub and LinkedIn. It highlights the importance of showcasing skills, projects, and academic background to build a strong digital portfolio for future opportunities.
- Module 3:
Explores platforms for coding practice and teamwork, such as HackerRank and Google Workspace tools. It helps students understand how to improve problem-solving skills and collaborate effectively in a digital environment.
- Module 4:
How to communicate professionally online, especially through emails. It emphasizes proper structure, tone, and etiquette, along with responsible behavior on social media platforms
- Module 5:
Focuses on identifying common cyber threats like phishing and UPI fraud. It also covers preventive measures and reporting mechanisms to ensure safe and secure online activities.
